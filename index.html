Introduction

Configurator Maven-Plugin (configurator-maven-plugin)

The configurator-maven-plugin is used to configure a JAR, WAR or EAR file after the build with properties.

Goals Overview

General information about the goals.

  • configurator:configure configure the target output of the current project.

  • configurator:configure-artifact configure an arbitrary artifact.

  • configurator:configure-file configures an external file.
