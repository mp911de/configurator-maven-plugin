Plugin Documentation

Goals available for this plugin:

Goal Description
configurator:configure Perform configuration for the project artifact.
configurator:configure-artifact Perform configuration for an external artifact.
configurator:configure-file Perform configuration an arbitrary, external file.
configurator:help Display help information on configurator-maven-plugin.
Call mvn configurator:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 2.0
JDK Default target for maven-compiler-plugin version 2.5.1
Memory No minimum requirement.
Disk Space No minimum requirement.

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>biz.paluch.maven.configurator</groupId>
          <artifactId>configurator-maven-plugin</artifactId>
          <version>1.0.1</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>biz.paluch.maven.configurator</groupId>
        <artifactId>configurator-maven-plugin</artifactId>
        <version>1.0.1</version>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>
